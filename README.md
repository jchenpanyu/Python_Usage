# matplotlib.pyplot
* **General Setting**
  * define the figure size (h, w)  
  *plt.figure(figsize=(h, w))*
  * set background grid  
  *plt.grid(True)*
  * set the x, y lim  
  *plt.xlim(x_low, x_up)*  
  *plt.ylim(y_low, y_up)*
  * set x,y label  
  *plt.xlabel('x_label)*
  *plt.ylabel('y_label)*
  * set plot title  
  *plt.title('title')*
